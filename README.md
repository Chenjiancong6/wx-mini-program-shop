# 从零到一，写了一个微信小程序购物商城项目。

## 项目的核心业务逻辑有：

- 商品搜索功能。
- 首页图片轮播。
- 商品分类页面、商品列表、商品详情页面。
- 商品加入购物车，购物添加/删除商品，获取微信内置收货地址。
- 微信登录模块，商品收藏模块。
- 小程序分享模块和联系客户模块。
